Connect to GitHub

Enter your GitHub Personal Access Token to start publishing blog posts. It's stored only in your browser — never sent anywhere except GitHub.

One-time setup (2 minutes):
1. Go to github.com/settings/tokens/new
2. Name it: Blog Admin
3. Expiration: No expiration (or 1 year)
4. Under Scopes check: repo
5. Click Generate token → copy and paste below
